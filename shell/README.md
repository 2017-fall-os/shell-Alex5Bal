# Shell

## Introduction
  -This program performs simple shell commands as directed by the user. 
## Functionality
  ### Creating Executable
    - Make: $ make
    - Run: $ ./shell
  ### Program Interaction
    - '$' signifies the program is ready to receive user input.
    - Program will keep prompting the user for input until 'X' is typed.
